# Cat Breed Classifier

This project is a simple cat breed image classifier built to learn more about Convolutional Neural Networks (CNNs) and deep learning for computer vision.

## Goals

- Learn how CNNs work
- Practice image preprocessing and augmentation
- Train and evaluate an image classification model
- Experiment with different network architectures and hyperparameters

## Dataset

The model will be trained on images of different cat breeds organized into labeled folders.
